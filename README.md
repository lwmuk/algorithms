# 算法练习 [![Build Status](https://travis-ci.org/lwmuk/algorithms.svg?branch=master)](https://travis-ci.org/lwmuk/algorithms)

## 简介
本项目主要是看《算法设计与分析基础（第3版）》时写的算法实现。

## 开发环境配置
### 克隆本项目
```bash
git clone https://github.com/lwmuk/algorithms.git
```
### 安装依赖包
```bash
cd algorithms
npm i
```
### 运行测试用例
```bash
npm t
```
